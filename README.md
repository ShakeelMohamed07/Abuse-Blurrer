# AI-Powered Toxic Comment Blurrer

A Chrome extension that detects and blurs toxic comments using a fine-tuned DistilBERT model.

## 🚀 Features
- Real-time toxic comment detection
- Automatic blurring of abusive content
- Toggle blur/unblur option
- Flask backend with ML model
- Chrome extension integration

## 🧠 Tech Stack
- Python (Flask)
- PyTorch
- Transformers (HuggingFace)
- JavaScript (Chrome Extension)
- HTML, CSS

## ⚙️ How to Run

1. Start backend:
   cd backend
   python app.py

2. Load extension:
   - Open Chrome → Extensions
   - Enable Developer Mode
   - Load unpacked → select project folder
